Interface control improvement modification:  Allow for remote configuration of the minium and maximum control values (e.g Minimum humidity to turn on the fan (Output #1))
Later these values can be displayed and configured using the greenhousegtkcontrol.py application.

To facilitate remote control value configuration the following constants would have to be stored in a file on disk or store in the greenhouse.db (SQlite3) database file.
